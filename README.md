## 🏫 Descripción

Sistema escolar diseñado para la **gestión académica y administrativa**, permitiendo el control de estudiantes, docentes, materias y procesos educativos dentro de una institución.

El sistema ofrece una interfaz intuitiva y herramientas que facilitan la organización y seguimiento de la información académica.

---

## 🚀 Características

- 📚 Gestión de estudiantes  
- 👨‍🏫 Administración de docentes  
- 🗂️ Control de materias  
- 📝 Registro de calificaciones  
- 📅 Gestión de horarios  
- 📊 Reportes académicos  

---

## 👥 Roles del Sistema

- 👨‍💼 Administrador  
  - Control total del sistema  
  - Gestión de usuarios y configuraciones  

- 👩‍🏫 Docente  
  - Registro de calificaciones  
  - Consulta de alumnos  

- 🎓 Estudiante  
  - Consulta de materias  
  - Visualización de calificaciones
